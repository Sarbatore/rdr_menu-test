## Red Dead Redemption 2 native menu for RedM.
Example menu of: https://github.com/Sarbatore/rdr_ui

Documentation: https://srb-documentation.gitbook.io/srbs-scripts/rdr_menu

Support: https://discord.gg/fxttPtFD66

## FEATURES:
### Menu
* Open
* Close
* SetHeader
* SetSubheader
* SetFooter
* SetFooterColor
* AddItem
* RemoveItem
* GetItem
* GetItemEntryId
* GetItems

### Menu Item
* SetText
* SetTextColor
* SetEnabled
* SetVisible
* OnFocused
* OnSelected

### Menu Image Item
* SetImgTextureDict
* SetImgTexture
